Download & Install

Version 1.1.0 was released on 1st of July, 2013. The latest version introduces amazing ability of WebDriver testing, now JMeter IS a browser :). There were also some other changes, see the changelog for full detail.

The prerequisite is to run JMeter 2.8 or above with a JRE 1.6. Follow installation procedure after download.

Usage

Once installed, you can use the plugins as standard JMeter objects, they will appear in the JMeter menus. You can save, copy to clipboard or export the graphs data by right-clicking them, and modify the rendering in different ways. All possibilities are described here.
